# Frontend Tienda E-commerce

Este repositorio contiene el frontend de una tienda de e-commerce. 

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5

## Funcionalidades
- Barra de navegación con links a secciones.
- Listado de productos en tarjetas.
- Botones para comprar y acceder al carrito.
- Página de detalle de cada producto.
- Footer con información de contacto y redes sociales.
